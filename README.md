# Nutcracker Nightwatch

A browser-playable 3D nutcracker prototype built with Three.js.

## Run

```powershell
npm.cmd install
npm.cmd run dev
```

Open http://127.0.0.1:5173/.

## Controls

- `W` / `ArrowUp`: move forward
- `S` / `ArrowDown`: move backward
- `A` / `ArrowLeft`: turn left
- `D` / `ArrowRight`: turn right
- Mouse movement after clicking the game: look / turn
- `Space` or mouse click: throw a bomb
- `Esc`: pause
